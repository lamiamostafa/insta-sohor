How Does Javascript work ?

Javascript is a high level and single threaded programming language. At first, the browser has loaded the page. The browser has been a built-in interpreter . That interpreter is read the code of Javascript. After that, it finds in the page and then runs it. Google chrome has its own javascript engine which name is v8 engine. Whenever we have run a JavaScript program inside a web browser, JavaScript code is being received by the browser's engine and then the engine has run the source code to obtain the output.

What does Javascript event loop do?

JavaScript has been a runtime model which is based on an event loop. For executing the code ,the event loop is used. The event loop is also used for collecting and processing the events. It is also used for executing queued sub-tasks.The event loop is mainly facilitating this process. The event loop is also constantly checking whether or not the call stack is empty. If it is empty, new functions has been added from the event queue. If it is not empty , then the current function call has been processed.
